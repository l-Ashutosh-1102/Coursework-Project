# FoodCorp Churn Prediction

## Business Problem

This project focuses on predicting customer churn using historical purchase behaviour and temporal customer activity features.

## Objective

The objective was to build a prediction system that identifies customers at risk of churn and supports marketing decision-making.

## Methods Used

- SQL-based feature engineering
- Temporal windowing
- Behavioural customer features
- Logistic regression
- Model evaluation
- Confusion matrix analysis
- Feature importance interpretation

## Tools Used

- SQL
- Python
- Pandas
- Scikit-learn
- Logistic Regression

## Key Output

The project produced a churn prediction model and business insights explaining which behavioural patterns were most associated with churn risk.
