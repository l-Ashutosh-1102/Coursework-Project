# Social Media Brand Analysis: Netflix vs Amazon Prime Video

## Business Problem

This project analyses public social media discussion to compare brand perception between Netflix and Amazon Prime Video.

## Objective

The objective was to identify sentiment patterns, key discussion themes, and potential micro-influencer accounts for brand engagement.

## Methods Used

- Text cleaning
- Sentiment analysis
- TF-IDF
- Topic modelling
- Brand comparison
- Micro-influencer recommendation

## Tools Used

- Python
- Pandas
- Natural Language Processing
- VADER
- TF-IDF
- NMF
- Matplotlib

## Key Output

The project provides brand-level insights on sentiment, topic themes, and influencer suitability.
