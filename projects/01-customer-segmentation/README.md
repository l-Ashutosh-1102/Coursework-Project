# Customer Segmentation for Retail Loyalty Data

## Business Problem

This project analyses customer transaction data to identify meaningful customer segments for marketing decision-making.

## Objective

The objective was to prepare customer-level features, reduce dimensionality, apply clustering, and interpret the resulting customer groups for commercial targeting.

## Methods Used

- Data cleaning
- Feature engineering
- Exploratory data analysis
- PCA
- K-Means clustering
- Segment profiling
- Marketing recommendation

## Tools Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Key Output

The project produced customer segments and recommended target groups for marketing action.
